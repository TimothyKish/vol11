# **Volume 5 Unified Lakes & Promotion Pipeline**

## **Overview**

Volume 5 is the *sovereign unification layer* of the KishLattice 16π Framework.  
It integrates domain‑specific scientific lakes into a single, geometry‑first, reproducible structure.

Each lake passes through three stages:

1. **Unified Scalarized Lake**  
2. **Promoted Lake (Vol5‑compliant)**  
3. **Validation**

Volume 5 also introduces two synthetic boundary lakes:

- **Planck Null Node** — the epsilon‑anchored geometric bottom  
- **Universal Closure Node** — the 16/π harmonic ceiling  

These define the lower and upper bounds of the scalar universe.

## **Lake Lifecycle**

### **1. Unified Scalarized Lake**

Upstream‑generated, geometry‑first, domain‑agnostic.  
Contains:

- entity_id  
- domain  
- volume  
- lake_id  
- geometry_payload  
- scalar_kls, scalar_klc  
- meta.source_row lineage  

Not yet Vol5‑compliant.

---

### **2. Promoted Lake**

Generated by `universal_promote.py`.

Adds:

- New UUID  
- Correct domain and volume  
- scalar_kls, scalar_klc (default 0.0 if missing)  
- Sovereign metadata:
  - meta.source = "vol5_promotion_raw_lake"
  - meta.ingest_timestamp
  - meta.sovereign = true
  - meta.source_row (full lineage)

Written to:

```
lakes/inputs_promoted/<lake>_promoted.jsonl
```

---

### **3. Validation**

`validate.py` ensures:

- Required fields exist  
- Geometry payload is complete  
- Meta fields are present  
- Domain matches volumes.json  
- No malformed entries  

A clean validation run guarantees reproducibility.

## **Extending the System**

### **Add a new lake**

1. Add unified scalarized file to `lakes/unified/`  
2. Add entry to `volumes.json`  
3. Promote  
4. Validate  

### **Add a new domain**

1. Add to schema.json (if needed)  
2. Add lakes to volumes.json  
3. Promote and validate

## **Troubleshooting**

### **Invalid domain**
Check spelling in `volumes.json`.

### **Missing scalar fields**
Ensure promoter is up to date.

### **Missing meta.source**
Promoter must set `"vol5_promotion_raw_lake"`.

### **Volume is a string**
Promoter must cast to integer.

### **Geometry missing fields**
Upstream scalarizer may be incomplete.
